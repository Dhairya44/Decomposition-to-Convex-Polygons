\doxysubsubsection{\texorpdfstring{main()}{main()}}
{\footnotesize\ttfamily main (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



The main function of the program.\+This function reads the input from the input.\+txt file and writes the output to the output.\+txt file. 

The input file should contain the number of vertices of the polygon and the coordinates of the vertices.

The output file contains the number of components of the polygon and the coordinates of the vertices of each component.

The time taken by the algorithm is also printed in the output file.

The algorithm is run on the polygon in clockwise order.

It takes the input and creates the \mbox{\hyperlink{class_d_c_e_l}{DCEL}} of the polygon.

It then calls the solve function to create subpolygons.

The solve function calls the merge functions to merge the subpolygons. \mbox{\Hypertarget{main_8cpp_a308376373e92f037c442bf5a05f45db8}\label{main_8cpp_a308376373e92f037c442bf5a05f45db8}}
